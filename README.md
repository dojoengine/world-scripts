# Basic JS Scripts for Katana Networks

This project was created using `bun init` in bun v1.0.25. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

To install dependencies:

```bash
bun install
```

To Create Account Details

```bash
bun run create
```

To Deploy Account

```bash
bun run deploy
```

To Check Balance

```bash
bun run deploy
```
